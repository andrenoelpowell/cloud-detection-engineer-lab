# Cloud Detection Pipeline

Example cloud detection pipeline:

AWS CloudTrail
      ↓
Log aggregation
      ↓
SIEM platform
      ↓
Detection rules
      ↓
Security alerts

Example tools used by organizations:

Cloud logs → AWS CloudTrail  
Log storage → S3 / log pipelines  
Detection engine → SIEM (Elastic, Splunk, Sentinel)  
Alerting → SOC or automated response
